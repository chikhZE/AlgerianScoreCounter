##  الميزات 
- دعم كامل لجميع شُعب البكالوريا والتعليم المتوسط.
- واجهات ديناميكية حديثة مبنية بـ **Jetpack Compose**.
- معالجة ذكية للمواد الاختيارية (مثل الأمازيغية والتربية البدنية).
- تصميم متناسق ومريح للعين يدعم الـ Dark Mode تلقائياً.

##  التقنيات المستخدمة
- **Kotlin** - اللغة الأساسية.
- **Jetpack Compose** - لبناء الواجهات (UI).
- **Navigation Compose** - للتنقل بين الشاشات.
- **Clean Architecture Principles** - لفصل البيانات عن منطق الواجهات.
